# Internshala Internship Search Clone

A responsive internship search page clone inspired by Internshala, built with Next.js.

## Features
- Internship listings fetched from Internshala API
- Frontend filtering by:
  - Profile
  - Location
  - Duration
  - Minimum stipend
- Work From Home filter
- Mobile responsive filter modal
- Clean UI inspired by Internshala

## Tech Stack
- Next.js
- React.js
- Tailwind CSS
